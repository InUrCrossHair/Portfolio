# Project images

Upload project preview images in this folder. The current project cards expect these file names:

- `small-business-marketing.png`
- `website-design-concepts.png`
- `business-development-materials.png`
- `event-promotion-graphics.png`
- `newsletter-success-story-layouts.png`
- `ai-workflow-experiments.png`

To use a different image name, update both the `src` and `data-full-image` values for that project card in `index.html`.
